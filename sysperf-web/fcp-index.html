Flex Consolidation Processor

Please see the Flex Consolidation Performance Analysis whitepaper for full details on how to use this web page for Flex consolidation. The uploaded files are expected to contain the binary sa files and a text file that contains additional appliance information.

How to use this page and limitations

This is a two step process: first upload the files and then process them. The uploaded files are not saved but deleted after processing. This is for security reasons. After uploading the files, you'll be presented with a button to process the uploaded files. Processing will start immediately but there will be no page updates until processing is completed. A link will be created for downloading the generated Excel spreadsheet upon processing completion.

If you need access to older data, simply reupload and reprocess the original files.

All fields below are required to be filled in.

  • This will support appliance code level 2.6.1.2 and later.
  • The customer name field will not accept any spaces. Use underscores or dashes instead.
  • After selecting files with the Browse button, you can hover over the Browse button to see list of files.
  • The following file size limits are being enforced:
    • Individual maximum file size = 1024 MB
    • Maximum number of files = 50
  • Each time you click on the Browse button, the previous file selections will be overwritten, i.e., it is not additive.
  • It takes approximately 10 seconds per uploaded zip file to process. Please be patient during the processing files phase.
